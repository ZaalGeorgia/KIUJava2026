# Student Handout

## Task

You are given a set of Java files.

Each file demonstrates one design pattern.

Your job:
1. identify the pattern
2. explain which classes play the important roles
3. justify your answer from the code structure

## Important Constraint

Class and method names were intentionally chosen to hide the pattern.

Do not guess from domain words.

You must justify the answer using:
- object relationships
- delegation
- inheritance
- runtime behavior

## Expected Output

For each file, write:
- file name
- guessed pattern
- 1-3 sentence justification

## Warning

Some examples are intentionally similar.

If you cannot explain why a pattern fits, then you have not identified it yet.
